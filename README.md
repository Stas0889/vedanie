# ВЕДАНИЕ landing

Статический лендинг без сборки.

## Как открыть

Откройте `index.html` в браузере.

## Локальный сервер

Если нужен preview через локальный сервер:

```bash
node dev-server.mjs
```

По умолчанию страница будет доступна на `http://127.0.0.1:4174/`.

## Перед публикацией

- Заменить CTA-якоря на реальные ссылки BotHelp / оплаты.
- Проверить desktop и mobile viewport.
- При необходимости заменить кропы из `assets/` на финальные изображения клиента.
